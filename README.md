# Athletic Agent Arbitrage (Beginner Version)

This beginner-friendly tool generates example athletic product opportunities.

## How to Run

1. Install Python: https://www.python.org/downloads/
2. Open Terminal or Command Prompt.
3. Navigate to this folder:
   cd athletic_agent_arbitrage
4. Run:
   python run_scan.py
5. View results in the outputs folder.

## Understanding the Results

- recommendation = "buy_review" → Strong opportunity
- recommendation = "manual_review" → Worth checking
- net_profit → Higher is better
- margin_percent → Aim for 18% or higher
- confidence → Prefer 0.90 or above
